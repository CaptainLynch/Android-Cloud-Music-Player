---

## 技术栈

```
UI:       Jetpack Compose + Material 3 (BOM 2026.02.01)
播放:     Media3 ExoPlayer 1.5.1
网络:     Retrofit 2.11 + OkHttp 4.12 + Gson
图片:     Coil Compose 2.7.0
异步:     Kotlin Coroutines 1.8.0 + Flow
持久化:   DataStore Preferences
构建:     Gradle Kotlin DSL, AGP 9.2.0, Kotlin 2.2.10
```

## API 概览

| 数据源 | 基础地址 | 用途 |
|--------|---------|------|
| GD Studio API | `https://music-api.gdstudio.xyz/api.php` | 搜索、播放 URL、封面、歌词 |
| 网易云 API | 自建 Binaryify 服务 / 直连 music.163.com（原生加密） | 登录、歌单、推荐、私人内容 |

---

## 开发规范

1. 纯 Kotlin + Compose，禁止 Java / XML 布局
2. MVVM + StateFlow 驱动，架构分层解耦
3. 不引入 Hilt/Dagger/Koin 等 DI 框架
4. 代码修改后更新对应角色文件的进度
5. 所有 AI 交互使用简体中文
